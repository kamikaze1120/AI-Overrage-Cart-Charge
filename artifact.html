AI Overage Cart Charge

This application calculates fees for overloaded bags based on user-uploaded data.

Features:

  • Upload Excel files containing bag counts.
  • Calculate fees for bags exceeding the defined threshold.
  • Analyze images related to the records.

How It Works:

  1. Upload an Excel file with columns RecordID and BagsCount.
  2. The application checks if the BagsCount exceeds the threshold.
  3. If it does, it calculates the fee based on the number of overloaded bags.
  4. Results are returned in JSON format.

Usage

To use the service, send a POST request to /upload with the file included.

Example Request:

    POST /upload
    Content-Type: multipart/form-data

    file: [your_excel_file.xlsx]

Response Format:

    [
        {
            "RecordID": 1,
            "Overloaded Bags": 6,
            "Fee Assigned": 10
        },
        ...
    ]
